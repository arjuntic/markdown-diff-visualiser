# Markdown Diff Review

When you modify any `.md` or `.markdown` file, remind the user that they can preview the rendered diff using the Markdown Diff Visualiser extension:

> You can see the rendered diff by running "Markdown Diff Visualiser: Show Changes" from the command palette (Cmd+Shift+P), or right-clicking the file in the explorer.

This is especially useful when:
- AI agents have modified documentation, READMEs, or changelogs
- Multiple sections of a markdown file were changed
- The changes include tables, images, code blocks, or other rich content that's hard to review in raw diff form

The extension shows a side-by-side rendered preview with green/red highlights on what changed, word-level precision, syntax-highlighted code blocks, and a scrollbar minimap.
